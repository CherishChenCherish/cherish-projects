Gold Monitor

Live — Running 24/7 via launchd

Overview

A production trading signal system for XAUUSD (gold) that runs autonomously on macOS. It monitors price in real-time, performs multi-factor technical analysis, calculates risk-reward ratios, and pushes actionable alerts to mobile devices via Bark.

Architecture

gold-api.com (30s poll) ↓ data_layer.py → OHLCV candles, ATR, volume, structure levels ↓ signal_engine.py → 5-factor scoring (0-9 pts) | • Multi-timeframe alignment (1m/5m/1h) | • RSI confirmation | • MACD alignment | • Volume analysis | • Bollinger Band position ↓ Score ≥ 4.5 AND R:R ≥ 2.0 → FIRE signal ↓ monitor_engine.py → Bark push + voice alert + dialog ↓ gold_web.py → Real-time dashboard (WebSocket)

Signal System

Multi-Factor Scoring

5 technical factors scored 0-9. Only signals scoring 4.5+ fire (6.0+ during NY session).

Risk Management

ATR-based stop losses (1.5x ATR). Minimum 2:1 risk-reward ratio enforced before any signal.

Structural Levels

Dynamic support, resistance, breakout, and demand zone tracking with auto-update.

Sentiment Overlay

News scoring, economic calendar suppression, and DXY correlation checks.

Key Technical Decisions

  • Backtested thresholds: MIN_SIGNAL_SCORE=4.5 and ATR_STOP_MULT=1.5 were optimized through historical data — not arbitrary
  • NY session gate: Score threshold raised to 6.0 during New York hours due to 15% win rate at lower thresholds
  • 15-minute cooldown: Prevents alert fatigue from repeated signals at the same level
  • Trade tracker: Logs every signal with entry, stop, target, score, and factors for post-analysis

Tech Stack

Python, FastAPI, WebSocket, gold-api.com, Bark push notifications, macOS launchd for process management, JSONL signal logging

Live Assistant

Beta — Core features complete

Overview

A real-time AI interview coach that listens to your interview via microphone, automatically detects when the interviewer asks a question, and streams a suggested answer to your screen — all in under 8 seconds.

Architecture

Microphone (sounddevice, 16kHz, 4s chunks) ↓ faster-whisper (CPU, int8 quantized) ↓ QuestionDetector | Stage 1: Pattern matching (45+ patterns) | Stage 2: LLM verification (weak signals only) ↓ Responder (Ollama HTTP streaming API) | • Behavioral: 3-4 sentence spoken response | • Technical: verbal + code + explanation ↓ asyncio.Queue → WebSocket (50ms poll) ↓ Frontend: streaming text display + card persistence

Features

Real-Time Transcription

Whisper-powered speech-to-text with VAD filtering. Supports system audio capture via BlackHole.

Smart Question Detection

45+ interview patterns with preamble extraction. Strong matches skip LLM for 5s faster response.

Streaming Responses

Answers appear token-by-token via Ollama streaming API. Cursor-pulse animation during generation.

Follow-Up Predictions

Predicts likely follow-up questions + generates keyword talking points asynchronously.

Context-Aware

Upload resume, job description, or notes. Responses reference your actual background.

Meeting App Detection

Auto-detects Zoom, Teams, WeMeet, Google Meet and recommends the correct audio device.

Key Technical Decisions

  • Thread-safe WebSocket bridge: Background threads use asyncio.Queue.put_nowait() to safely send messages to the async event loop
  • Ollama HTTP API over subprocess: Saves 2-3s per call from process startup overhead, enables streaming
  • Response length cap (350 chars): Optimized for interview context — must be readable in 5 seconds
  • Model warmup on startup: Sends a 1-token request to pre-load the model into GPU/CPU memory

Tech Stack

Python, FastAPI, WebSocket, faster-whisper, Ollama (gemma3:4b), sounddevice, HMAC-SHA256 license system, Apple-inspired UI

AI-PM

In Development

Overview

An AI-native project management application that combines traditional PM features (tasks, goals, time tracking) with an integrated AI assistant for planning, prioritization, and weekly reviews.

Architecture

React + Vite (port 5173) ↓ REST API FastAPI (port 8000) | • /api/projects, /api/tasks, /api/goals | • /api/time-logs, /api/calendar | • /api/reviews, /api/ai ↓ SQLite + aiosqlite (async ORM) ↓ AI Client (Ollama or OpenAI — configurable)

Features

Project Management

Full CRUD for projects, tasks with status tracking, and hierarchical organization.

Goal Tracking

Set goals with deadlines, track progress, and review completion rates.

Time Logging

Track time spent on tasks with detailed logs and aggregated reports.

AI Assistant

Integrated chat for planning, task breakdown, and weekly review generation. Supports Ollama and OpenAI.

Calendar Integration

Calendar events with iCal utilities for scheduling and deadline management.

Scheduled Tasks

Background scheduler for recurring notifications and automated reviews.

Tech Stack

FastAPI, React, Vite, SQLite + aiosqlite, Alembic migrations, Ollama / OpenAI, APScheduler

Development Status

  • Backend: Complete — all API endpoints functional, database schema finalized
  • Frontend: Complete — built and bundled with Vite
  • AI integration: Working with Ollama, OpenAI support configured but untested in production
  • Deployment: Local only — not yet deployed to a public server
